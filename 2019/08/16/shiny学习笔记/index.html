shiny中的常用功能调用方法

shiny各种小部件

I. 介绍

说说有哪些常用部件。
- actionButton：动作按钮
- checkboxGroupInput：复选框组，多选框，打钩的
- checkboxInput：单个复选框，单个打钩的
- dateInput：日期选择控件
- dateRangeInput：日期区间选择控件
- fileInput：文件上传控件
- helpText：帮助文档，类似word里面的文本框
- numericInput：数值输入框
- radioButtons：单选按钮，多个选项但只能选一个，打点的
- selectInput：下拉选框，待选盒
- sliderInput：滑动条
- submitButton：提交按钮
- textInput：文本输入框

II. 使用方法
小部件的函数中有很多选项，如函数column()也有多个参数。
- column(width, ..., offset = 0)函数：第一个参数是部件宽度，整个页面共宽12(单位：column)；中间... 是加入的部件函数、部件显示的title等等；最后参数offset指部件向右偏倚多少column
- 部件名称：一般是部件函数的第一个参数，我们不会看到，但是就相当于变量名一样，在之后的server.R中能够通过这个调用该部件的选择或输入结果
- 部件标签（label）：会与部件一起显示，必须是string，但可为空
- 寻求帮助：和所有函数一样使用?就能得到帮助，如?selectInput

III. 示例
以下为介绍的12个部件函数的使用：
shinyUI(fluidPage(
  titlePanel("Basic widgets"),

  fluidRow(                                                     # 第一行
    column(3,                                                  # 第一个部件宽度为3 col
           h3("Buttons"),                                     # 显示部件title，有的在部件函数里面可以调用
           actionButton("action", label="Action"),  # 动作按钮，变量名为action，标签为Action
           br(),                                                   # 空行
           br(),
           submitButton("Submit")),                     # 第二个按钮，提交按钮
    column(3,
           h3("Single check"),                             # 部件title
           checkboxInput("checkbox", label="Choice A", value=TRUE)),    
           # 单个复选框：部件名称，标签，默认值TRUE=选定
    column(3,
           checkboxGroupInput("checkGroup",               # 复选框组：部件名称
                              label=h3("Checkbox group"),       # 部件标签
                              choices=list("Choice 1"=1,          # 选项1与对应取值1
                                                "Choice 2"=2,          # 选项2与对应取值2
                                                "Choice 3"=3),         # 选项3与对应取值3
                              selected=1)),                             # 默认选项为选项1
    column(3,
           dateInput("date",                                           # 日期输入：名称date
                     label=h3("Date input"),
                     value="2014-01-01"))                           # 默认输入日期
  ),
  fluidRow(                                                                       # 第二行
    column(3,
           dateRangeInput("dates", label=h3("Date range"))),   # 日期区间
    column(3,
           fileInput("file", label=h3("File input"))),                    # 文件上传
    column(3,
           h3("Help text"),                                                    # 部件title
           helpText("Note: help text isn't a true widget,",        # helptext的内容，可分行
                    "but it provides an easy way to add text to",
                    "accompany other widgets.")),
    column(3,
           numericInput("num",                                             # 数值输入
                        label=h3("Numeric input"),                       # 部件title
                        value=1))                                                # 默认值1
  ),
  fluidRow(                                                                      # 第三行
    column(3,
           radioButtons("radio", label=h3("Radio buttons"),      # 单选框：名称和标签/title
                        choices=list("Choice 1"=1,                       # 三个选项与对应的取值
                                          "Choice 2"=2,
                                          "Choice 3"=3),
                        selected=1)),                                          # 默认选项1
    column(3,
           h3("Select input"),                                                # 部件title
           selectInput("variable", "Variable:",                         # 下拉框：名称，标签
                       c("Cylinders" = "cyl",                               # 下拉的内容和对应的取值
                         "Transmission" = "am",
                         "Gears" = "gear"))),
    column(3,
           sliderInput("slider1", label=h3("Sliders"),               # 滑条：名称和标签title
                       min=0, max=100, value=50),                   # 最大值，最小值和默认值
           sliderInput("slider2", "",                                       # 双向滑条：名称，无标签
                       min=0, max=100, value=c(25, 75))),         # 最大值，最小值，默认区间值
    column(3,
           textInput("text", label=h3("Text input"),                 # 文本输入框：名称和标签
                     value="Enter text..."))                               # 默认输入值
  )
))
